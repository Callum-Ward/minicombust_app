# miniCOMBUST

## Dependencies
- Catch++ Unit Testing (Header-only)


## Build

```
make clean all
```

## Run 
```
./bin/minicombust
```

## Run tests 
```
./bin/minicombust_tests
```

## Unsupported Feature List
- Spray Breakup (spawn new particles, primary and secondary breakup)
- C API for particle side
- Tetrahedral mesh
- Benchmarks
- Serial Optimised Implementation 
- MPI Implementation
